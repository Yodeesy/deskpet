DeskFox

Fox rules the world.

State: Idle
Description: 狐狸在没人打扰的时候喜欢安静地思考。虽然无论怎么思考，这个世界都是这样。

State: dragging
Description:你抓起了狐狸。狐狸只能挣扎或者表现出一副生无可恋的样子 (毕竟谁被抓到半空中都不舒服吧喂!)

State: angry
Description: 你过于频繁地抓起狐狸。狐狸很生气。

State: upset
Description: 你一段时间未与狐狸互动。狐狸不想打扰你。于是躲在角落里哭泣。

State: fishing
Description: 狐狸喜欢钓鱼。虽然每次钓上来的都是漂流瓶...

State: teleport
Description: 狐狸施展瞬间移动法术 (拜托千万不要卡在墙里啊!)

State: magic
Description: 你不会以为狐狸要抱你吧？答案是狐狸正在施展法术...效果是什么呢？

State: butterfly
Description: 其实狐狸并不想要抓住蝴蝶，它们是自由的生命。狐狸只是想要陪你玩。
